# Система управления библиотекой

## Функции

- Добавление книг в библиотеку
- Удаление книг из библиотеки
- Изменение статуса книги ("в наличии" или "выдана")
- Поиск книги по ID
- Отображение всех книг
- Загрузка/выгрузка библиотеки из/в JSON файл

## Установка

1. Установите репозиторий:
```
git clone https://github.com/1ef7yy/effective_mobile_test.git
```

2. Запустите приложение:
```
python main.py
```

## Формат JSON файла для сохранения библиотеки
```json
{
    "books": [
        {
            "title": "Паттерны Объектно-ориентированного Программирования",
            "author": "Э. Гамма, Р. Хелм",
            "year": 2020,
            "status": "в наличии",
            "id": 1
        }
    ]
}
```
